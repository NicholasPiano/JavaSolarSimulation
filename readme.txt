How to use:

1. Create Folders in src directory

JupiterDataIn
JupiterDataOut
MarsDataIn
MarsDataOut

2. Compile all java src files

javac *.java (or whichever compiler you are using)

3. Run python controllers

python JupiterRunner.py initial_x initial_vx initial_y initial_vy
python MarsRunner.py '' '' '' ''

4. Enjoy

Terminal output should give distance of closest approach of the satellites
